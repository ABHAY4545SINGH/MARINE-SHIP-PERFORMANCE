Marine Metrics

Data-Driven Analysis of Ship Performance

About the Project

This project focuses on improving ship performance and operational efficiency using data analytics and machine learning. Through predictive modeling, clustering, and interactive dashboards, Marine Metrics provides actionable insights for route planning, engine optimization, and maintenance scheduling.

Objectives

  • Perform EDA to identify trends in maritime operations.
  • Analyze fuel efficiency, cost, and revenue across ship types.
  • Build clustering models for ship segmentation.
  • Develop predictive models for cost and efficiency estimation.
  • Visualize insights through a real-time interactive dashboard.

Project Modules

Includes data ingestion, cleaning, EDA, clustering, predictive modeling, dashboard UI (Flask), and report generation.
